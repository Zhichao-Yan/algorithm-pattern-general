### 先序遍历
* 递归方法
```C++ 
    void  PreOrderTraverse(TreeNode* root)
    {
        if(!root)
            return;
        else{
            visit(root);
            Traverse(root->left);
            Traverse(root->right);
        }
    }
```
### 中序遍历
* 递归方法
```C++ 
    void  InOrderTraverse(TreeNode* root)
    {
        if(!root)
            return;
        else{
            Traverse(root->left);
            visit(root);
            Traverse(root->right);
        }
    }
```

### 后序遍历
* 递归方法
```C++ 
    void  PostOrderTraverse(TreeNode* root)
    {
        if(!root)
            return;
        else{
            Traverse(root->left);
            Traverse(root->right);
            visit(root);
        }
    }
```
