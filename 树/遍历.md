### 先序遍历
* 递归方法
```C++ 
    void  Traverse(TreeNode* root)
    {
        if(!root)
            return;
        else{
            cout<<root->val;
            Traverse(root->left);
            Traverse(root->right);
        }
    }
```
### 中序遍历
* 递归方法
```C++ 
    void  Traverse(TreeNode* root)
    {
        if(!root)
            return;
        else{
            Traverse(root->left);
            cout<<root->val;
            Traverse(root->right);
        }
    }
```

### 后序遍历
* 递归方法
```C++ 
    void  Traverse(TreeNode* root)

        if(!root)

            Traverse(root->left);
            Traverse(root->right);
```
